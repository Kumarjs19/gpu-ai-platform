# GPU AI SaaS Platform

🚀 Production-ready AI SaaS platform

## Features
- FastAPI backend
- GPU workers (CUDA Docker)
- Redis queue
- Multi-user authentication (JWT)
- Razorpay UPI payments
- Credits system
- Usage analytics

## Architecture
UI → API → Redis → GPU Workers

## Quick Start
```bash
docker-compose up --build
```

## Modules
- /api → Backend
- /worker → GPU execution
- /ui → Frontend
